# Tic-Tac-Toe PHP project

## Requirements

- [docker](https://www.docker.com/get-started)
- [docker-compose](https://docs.docker.com/compose/install/)

## Install

```
make install
```

## Run

```
make start
```

Runs on http://localhost:3000

## Install new dependency

```
make required slim/slim "^3.0"
```
